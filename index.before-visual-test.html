遺品整理（ご逝去後）

ご逝去後の整理は、家財を減らす前に「探す物」「残す物」「相談する物」を分けると失敗しにくくなります。特に賃貸退去や売却前は急ぎがちですが、書類・貴重品・形見分け品を先に確認してください。

1. 期限退去日、鍵返却日、売却査定日、法要の日程
2. 探す物通帳、印鑑、権利書、保険証券、契約書、遺言書
3. 残す物写真、手紙、形見分け品、仏壇、位牌、供養品
4. 任せる範囲仕分け、搬出、買取、供養、清掃、写真報告
  • 封のある遺言書を見つけたら、自己判断で開封せず専門窓口へ確認
  • 相続税の申告が必要な場合は、死亡を知った日の翌日から10か月以内が期限
  • 家庭ごみ・粗大ごみは市区町村ルールや許可業者の確認が必要
  • 見積もりでは、間取りだけでなく荷物量、階数、駐車位置、搬出経路を共有
  • 買取希望品は処分品と分け、貴金属・時計・家電・趣味用品を先に確認
  • 遠方の場合は、鍵預かり、作業前後写真、貴重品返送、精算方法を確認
先に捨てない方がよい物：戸籍・権利書・固定資産税通知書・保険証券・年金書類・契約書・写真・手紙・印鑑・通帳・貸金庫や金庫の鍵。

孤独死・特殊清掃

孤独死後は、通常の家財整理だけでなく、室内状況の確認、消臭、汚染箇所の撤去、原状回復の相談が必要になることがあります。

1. 入室状況発見日、警察確認、入室可否、鍵の所在
2. 室内状態臭気、汚染箇所、床材、畳、害虫の有無
3. 関係者大家、管理会社、保険会社、親族への報告
4. 作業範囲特殊清掃、家財搬出、消臭、原状回復前確認
  • 作業前後の写真報告が出るか確認
  • 汚染箇所の撤去範囲と追加費用条件を見積書に残す
  • 家財整理と特殊清掃を同日で進められるか確認
  • 貴重品・書類探索を処分前に依頼する
  • 近隣や共用部への臭気対策、搬出時間を確認
  • 管理会社へ提出できる作業記録の有無を確認
注意：通常の遺品整理より緊急性と専門性が高くなります。電話だけで確定金額にせず、室内状況・撤去範囲・消臭範囲を分けて確認してください。

生前整理

生前整理は、家を空にする作業ではありません。本人が元気なうちに、残す物、譲る物、処分する物を家族と一緒に決めていく準備です。

1. 残す物通帳、印鑑、保険証券、年金書類、写真
2. 減らす場所玄関、廊下、寝室、台所、押し入れ
3. 譲る物形見分け予定品、家具、着物、趣味用品
4. 任せる作業仕分け補助、搬出、買取、供養、簡易清掃
  • 本人の判断を優先し、家族だけで処分を決めない
  • 生活動線に関わる場所から始める
  • 重要書類の保管場所を家族に共有する
  • 買取希望品と譲る品を混ぜない
  • 仏壇・神棚・人形・写真の扱いを先に決める
  • 一度で全室ではなく、部屋ごとに分けて進める
進め方：最初は「捨てる」より「分ける」が大切です。残す・譲る・売る・供養する・処分する、の5分類にすると家族間の行き違いを減らせます。

老人ホーム・施設入居前の整理

施設入居前は、入居日までに持ち込む荷物を絞り、実家に残す家具や処分する家財を決める必要があります。

1. 入居日搬出日、鍵管理、退去・売却予定
2. 持込品衣類、寝具、日用品、薬、写真、書類
3. 実家の扱い空き家管理、売却、解体、親族利用
4. 残置物大型家具、家電、物置、仏壇、貴重品
  • 施設の収納量と持ち込み禁止品を確認
  • 介護用品・薬・診察券を処分品に混ぜない
  • 本人が使う写真や日用品を先に確保
  • 実家へ戻る可能性があるか家族で確認
  • 大型家具・家電の搬出経路を見積もり前に共有
  • 入居日直前に慌てないよう、残す物を先に別室へまとめる
注意：施設入居前は時間が限られます。持ち込む物の箱と、処分・買取・保留の箱を分けるだけでも当日の混乱を減らせます。

実家じまい・空き家（相続）

相続後の実家整理では、家財を片付ける前に、権利書、固定資産税通知書、保険証券、契約書などを探す必要があります。

1. 書類権利書、固定資産税通知書、契約書、保険証券
2. 家の予定売却、解体、賃貸、空き家管理、親族利用
3. 範囲室内、庭、物置、車庫、納屋、ベランダ
4. 供養品仏壇、神棚、位牌、写真、人形、手紙
  • 売却査定日や解体予定日から逆算する
  • 庭・物置・車庫の残置物も見積もり範囲に入れる
  • 権利書や固定資産税関係書類を先に探す
  • 不動産会社へ残置物の扱いを確認
  • 家電リサイクル対象品の処分方法を確認
  • 遠方の場合は写真報告と鍵の返却方法を決める
先に確認：空き家は室内だけでなく、庭・物置・車庫で費用が変わります。見積もり前に写真を撮っておくと、追加費用の判断がしやすくなります。

賃貸退去に伴う整理

賃貸退去では、退去日と鍵返却日が決まっているため、短期間で仕分け、搬出、処分、簡易清掃まで進める必要があります。

1. 期限退去日、鍵返却日、立会い日、清掃日
2. 建物条件階数、エレベーター、養生、駐車位置
3. 残す物貴重品、書類、形見分け品、買取希望品
4. 管理会社搬出時間、共用部ルール、粗大ごみ置場
  • 鍵返却日までに全量搬出できるか確認
  • エレベーター予約や共用部養生の要否を確認
  • 作業可能時間と駐車位置を管理会社へ確認
  • 残置物扱いになる物を残さない
  • 退去立会い前に簡易清掃まで必要か確認
  • 急ぎの場合も、追加料金条件を見積書に残す
注意：賃貸退去は日程遅れが負担になりやすいです。退去日ではなく「鍵返却日」から逆算して、搬出・清掃・確認の順番を決めてください。

5. サービスの違い

「遺品整理」「生前整理」「特殊清掃」「不用品回収」は似ていますが、確認する内容が違います。目的に近いサービス名を知っておくと、見積もり時の話が早くなります。

サービス対象タイミング主な作業向いているケース
遺品整理ご逝去後、退去・相続・売却前仕分け、貴重品探索、搬出、買取査定、供養品の分別、写真報告通帳・印鑑・権利書・写真を探しながら、家財を整理したい
生前整理本人が元気なうち、施設入居前必要品の選別、不要品の搬出、書類整理、家族への共有、買取子どもに負担を残したくない、使わない家具や家電を減らしたい
特殊清掃孤独死、事故、室内汚染がある時消臭、汚染箇所の撤去、原状回復前の確認、家財整理大家・管理会社対応や原状回復を含めて急いで進めたい
不用品回収引越し、買い替え、少量処分家具・家電・粗大ごみの搬出、回収、簡易仕分け探し物や供養は不要で、処分したい物が決まっている

6. 費用相場の全体像

料金は間取りだけで決まりません。荷物量、階数、搬出距離、駐車位置、買取品、供養品の有無で変わります。

2DK70,000円〜家具・家電・衣類・書類、共用部養生、搬出経路。

料金は、荷物量、階数、エレベーターの有無、搬出経路、駐車位置、買取の有無、供養品の量で変動します。見積もり時は、写真だけでなく「押入れ・物置・ベランダ・庭」も共有すると差が出にくくなります。

7. 業者選び・トラブル回避の基本

安さだけで選ぶと、追加料金、雑な搬出、大切な品の誤処分につながることがあります。

  • 見積書に作業範囲、処分量、追加料金条件が書かれているか
  • 買取を行う場合、古物商許可や査定内容を確認できるか
  • 一般廃棄物の扱いを自治体ルールに沿って説明できるか
  • 作業前後の写真報告、鍵預かり、貴重品返却の方法が明確か
  • 相見積もりを嫌がらず、比較に必要な内訳を出してくれるか

8. よくある質問

どの地域ページを見ればよいですか？

まず実家や作業場所に近い都道府県を選び、次に市区町村・区ページを確認してください。

生前整理は何から始めればよいですか？

玄関、寝室、通路など生活の安全に関わる場所から始めると進めやすいです。

遠方で立会いできなくても進められますか？

鍵預かり、写真報告、オンライン確認、振込精算を組み合わせると進めやすくなります。

孤独死後の特殊清掃も一緒に相談できますか？

相談前に、入室可否、室内状況、消臭の必要性、大家・管理会社への報告範囲を整理してください。通常の遺品整理とは作業内容が変わります。

老人ホーム入居前の片付けは何を確認しますか？

入居日、施設に持ち込む衣類・寝具・日用品、実家に残す家具、処分する家財、空き家管理や売却予定を確認してください。

賃貸退去に伴う整理は急ぎでも相談できますか？

退去日、鍵返却日、エレベーター利用、共用部養生、搬出時間、駐車位置を先に共有すると、短期対応の可否を確認しやすくなります。

買取や供養も相談できますか？

家電、貴金属、時計、趣味用品と、供養したい仏壇、神棚、写真、人形を分けて伝えてください。
